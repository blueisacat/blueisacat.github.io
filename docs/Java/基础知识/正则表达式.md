从url中截取域名

```
(?<=http(s?)://)([^/]*?)(?=[/:])|(?<=http(s?)://)([^/]*)(?=[/:])
```

从url中截取路径

```
(?<=(http(s?)://.*/))(.*)
```

从url中截取上级域名

```
(?<=(http(s?)://[^/]*?\.))(.*?)(?=/)|(?<=(http(s?)://[^/]*?\.))(.*)
```

除去所有标点符号

```
base = base.replaceAll("[\\pP]", " ");
```

出去所有空格

```
base = base.replaceAll("[\\s\\p{Zs}]+", " ").trim();
```
