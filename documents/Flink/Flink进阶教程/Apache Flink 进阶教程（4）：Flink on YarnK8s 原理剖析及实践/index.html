    1 Flink жһ¶жһ„жҰӮи§Ҳ¶

    1.1 Flink жһ¶жһ„жҰӮи§ҲвҖ“Job¶

    з”ЁжҲ·йҖҡиҝҮ DataStream API гҖҒ DataSet API гҖҒ SQL е’Ң Table API зј–еҶҷ Flink д»»еҠЎпјҢе®ғдјҡз”ҹжҲҗдёҖдёӘ JobGraph гҖӮ JobGraph жҳҜз”ұ source гҖҒ map() гҖҒ keyBy()/window()/apply() е’Ң Sink зӯүз®—еӯҗз»„жҲҗзҡ„гҖӮеҪ“ JobGraph жҸҗдәӨз»ҷ Flink йӣҶзҫӨеҗҺпјҢиғҪеӨҹд»Ҙ Local гҖҒ Standalone гҖҒ Yarn е’Ң Kubernetes еӣӣз§ҚжЁЎејҸиҝҗиЎҢгҖӮ

    1.2 Flink жһ¶жһ„жҰӮи§ҲвҖ“JobManager¶

    JobManager зҡ„еҠҹиғҪдё»иҰҒжңүпјҡ

    • е°Ҷ JobGraph иҪ¬жҚўжҲҗ Execution Graph пјҢжңҖз»Ҳе°Ҷ Execution Graph жӢҝжқҘиҝҗиЎҢ

    • Scheduler з»„д»¶иҙҹиҙЈ Task зҡ„и°ғеәҰ

    • Checkpoint Coordinator з»„д»¶иҙҹиҙЈеҚҸи°ғж•ҙдёӘд»»еҠЎзҡ„ Checkpoint пјҢеҢ…жӢ¬ Checkpoint зҡ„ејҖе§Ӣе’Ңе®ҢжҲҗ

    • йҖҡиҝҮ Actor System дёҺ TaskManager иҝӣиЎҢйҖҡдҝЎ

    • е…¶е®ғзҡ„дёҖдәӣеҠҹиғҪпјҢдҫӢеҰӮ Recovery Metadata пјҢз”ЁдәҺиҝӣиЎҢж•…йҡңжҒўеӨҚж—¶пјҢеҸҜд»Ҙд»Һ Metadata йҮҢйқўиҜ»еҸ–ж•°жҚ®гҖӮ

    1.3 Flink жһ¶жһ„жҰӮи§ҲвҖ“TaskManager¶

    TaskManager жҳҜиҙҹиҙЈе…·дҪ“д»»еҠЎзҡ„жү§иЎҢиҝҮзЁӢпјҢеңЁ JobManager з”іиҜ·еҲ°иө„жәҗд№ӢеҗҺејҖе§ӢеҗҜеҠЁгҖӮ TaskManager йҮҢйқўзҡ„дё»иҰҒз»„д»¶жңүпјҡ

    • Memory & I/O Manager пјҢеҚіеҶ…еӯҳ I/O зҡ„з®ЎзҗҶ

    • Network Manager пјҢз”ЁжқҘеҜ№зҪ‘з»ңж–№йқўиҝӣиЎҢз®ЎзҗҶ

    • Actor system пјҢз”ЁжқҘиҙҹиҙЈзҪ‘з»ңзҡ„йҖҡдҝЎ

    TaskManager иў«еҲҶжҲҗеҫҲеӨҡдёӘ TaskSlot пјҢжҜҸдёӘд»»еҠЎйғҪиҰҒиҝҗиЎҢеңЁдёҖдёӘ TaskSlot йҮҢйқўпјҢ TaskSlot жҳҜи°ғеәҰиө„жәҗйҮҢзҡ„жңҖе°ҸеҚ•дҪҚгҖӮ

    еңЁд»Ӣз»Қ Yarn д№ӢеүҚе…Ҳз®ҖеҚ•зҡ„д»Ӣз»ҚдёҖдёӢ Flink Standalone жЁЎејҸпјҢиҝҷж ·жңүеҠ©дәҺжӣҙеҘҪең°дәҶи§Ј Yarn е’Ң Kubernetes жһ¶жһ„гҖӮ

    • еңЁ Standalone жЁЎејҸдёӢпјҢ Master е’Ң TaskManager еҸҜд»ҘиҝҗиЎҢеңЁеҗҢдёҖеҸ°жңәеҷЁдёҠпјҢд№ҹеҸҜд»ҘиҝҗиЎҢеңЁдёҚеҗҢзҡ„жңәеҷЁдёҠгҖӮ

    • еңЁ Master иҝӣзЁӢдёӯпјҢ Standalone ResourceManager зҡ„дҪңз”ЁжҳҜеҜ№иө„жәҗиҝӣиЎҢз®ЎзҗҶгҖӮеҪ“з”ЁжҲ·йҖҡиҝҮ Flink Cluster Client е°Ҷ JobGraph жҸҗдәӨз»ҷ Master ж—¶пјҢ JobGraph е…Ҳз»ҸиҝҮ Dispatcher гҖӮ

    • еҪ“ Dispatcher ж”¶еҲ°е®ўжҲ·з«Ҝзҡ„иҜ·жұӮд№ӢеҗҺпјҢз”ҹжҲҗдёҖдёӘ JobManager гҖӮжҺҘзқҖ JobManager иҝӣзЁӢеҗ‘ Standalone ResourceManager з”іиҜ·иө„жәҗпјҢжңҖз»ҲеҶҚеҗҜеҠЁ TaskManager гҖӮ

    • TaskManager еҗҜеҠЁд№ӢеҗҺпјҢдјҡжңүдёҖдёӘжіЁеҶҢзҡ„иҝҮзЁӢпјҢжіЁеҶҢд№ӢеҗҺ JobManager еҶҚе°Ҷе…·дҪ“зҡ„ Task д»»еҠЎеҲҶеҸ‘з»ҷиҝҷдёӘ TaskManager еҺ»жү§иЎҢгҖӮ

    д»ҘдёҠе°ұжҳҜдёҖдёӘ Standalone д»»еҠЎзҡ„иҝҗиЎҢиҝҮзЁӢгҖӮ

    1.4 Flink иҝҗиЎҢж—¶зӣёе…із»„д»¶¶

    жҺҘдёӢжқҘжҖ»з»“дёҖдёӢ Flink зҡ„еҹәжң¬жһ¶жһ„е’Ңе®ғеңЁиҝҗиЎҢж—¶зҡ„дёҖдәӣз»„д»¶пјҢе…·дҪ“еҰӮдёӢпјҡ

    • Client пјҡз”ЁжҲ·йҖҡиҝҮ SQL жҲ–иҖ… API зҡ„ж–№ејҸиҝӣиЎҢд»»еҠЎзҡ„жҸҗдәӨпјҢжҸҗдәӨеҗҺдјҡз”ҹжҲҗдёҖдёӘ JobGraph гҖӮ

    • JobManager пјҡ JobManager жҺҘеҸ—еҲ°з”ЁжҲ·зҡ„иҜ·жұӮд№ӢеҗҺпјҢдјҡеҜ№д»»еҠЎиҝӣиЎҢи°ғеәҰпјҢе№¶дё”з”іиҜ·иө„жәҗеҗҜеҠЁ TaskManager гҖӮ

    • TaskManager пјҡе®ғиҙҹиҙЈдёҖдёӘе…·дҪ“ Task зҡ„жү§иЎҢгҖӮ TaskManager еҗ‘ JobManager иҝӣиЎҢжіЁеҶҢпјҢеҪ“ TaskManager жҺҘж”¶еҲ° JobManager еҲҶй…Қзҡ„д»»еҠЎд№ӢеҗҺпјҢејҖе§Ӣжү§иЎҢе…·дҪ“зҡ„д»»еҠЎгҖӮ

    2 Flink on Yarn еҺҹзҗҶеҸҠе®һи·ө¶

    2.1 Yarn жһ¶жһ„еҺҹзҗҶвҖ“жҖ»и§Ҳ¶

    Yarn жЁЎејҸеңЁеӣҪеҶ…дҪҝз”ЁжҜ”иҫғе№ҝжіӣпјҢеҹәжң¬дёҠеӨ§еӨҡж•°е…¬еҸёеңЁз”ҹдә§зҺҜеўғдёӯйғҪдҪҝз”ЁиҝҮ Yarn жЁЎејҸгҖӮйҰ–е…Ҳд»Ӣз»ҚдёҖдёӢ Yarn зҡ„жһ¶жһ„еҺҹзҗҶпјҢеӣ дёәеҸӘжңүи¶іеӨҹдәҶи§Ј Yarn зҡ„жһ¶жһ„еҺҹзҗҶпјҢжүҚиғҪжӣҙеҘҪзҡ„зҹҘйҒ“ Flink жҳҜеҰӮдҪ•еңЁ Yarn дёҠиҝҗиЎҢзҡ„гҖӮ

    Yarn зҡ„жһ¶жһ„еҺҹзҗҶеҰӮдёҠеӣҫжүҖзӨәпјҢжңҖйҮҚиҰҒзҡ„и§’иүІжҳҜ ResourceManager пјҢдё»иҰҒз”ЁжқҘиҙҹиҙЈж•ҙдёӘиө„жәҗзҡ„з®ЎзҗҶпјҢ Client з«ҜжҳҜиҙҹиҙЈеҗ‘ ResourceManager жҸҗдәӨд»»еҠЎгҖӮ

    з”ЁжҲ·еңЁ Client з«ҜжҸҗдәӨд»»еҠЎеҗҺдјҡе…Ҳз»ҷеҲ° Resource Manager гҖӮ Resource Manager дјҡеҗҜеҠЁ Container пјҢжҺҘзқҖиҝӣдёҖжӯҘеҗҜеҠЁ Application Master пјҢеҚіеҜ№ Master иҠӮзӮ№зҡ„еҗҜеҠЁгҖӮеҪ“ Master иҠӮзӮ№еҗҜеҠЁд№ӢеҗҺпјҢдјҡеҗ‘ Resource Manager еҶҚйҮҚж–°з”іиҜ·иө„жәҗпјҢеҪ“ Resource Manager е°Ҷиө„жәҗеҲҶй…Қз»ҷ Application Master д№ӢеҗҺпјҢ Application Master еҶҚе°Ҷе…·дҪ“зҡ„ Task и°ғеәҰиө·жқҘеҺ»жү§иЎҢгҖӮ

    2.2 Yarn жһ¶жһ„еҺҹзҗҶвҖ“з»„д»¶¶

    Yarn йӣҶзҫӨдёӯзҡ„з»„д»¶еҢ…жӢ¬пјҡ

    • ResourceManager (RM) пјҡ ResourceManager (RM) иҙҹиҙЈеӨ„зҗҶе®ўжҲ·з«ҜиҜ·жұӮгҖҒеҗҜеҠЁжҲ–зӣ‘жҺ§ ApplicationMaster гҖҒзӣ‘жҺ§ NodeManager гҖҒиө„жәҗзҡ„еҲҶй…ҚдёҺи°ғеәҰпјҢеҢ…еҗ« Scheduler е’Ң Applications Manager гҖӮ

    • ApplicationMaster (AM) пјҡ ApplicationMaster (AM) иҝҗиЎҢеңЁ Slave дёҠпјҢиҙҹиҙЈж•°жҚ®еҲҮеҲҶгҖҒз”іиҜ·иө„жәҗе’ҢеҲҶй…ҚгҖҒд»»еҠЎзӣ‘жҺ§е’Ңе®№й”ҷгҖӮ

    • NodeManager (NM) пјҡ NodeManager (NM) иҝҗиЎҢеңЁ Slave дёҠпјҢз”ЁдәҺеҚ•иҠӮзӮ№иө„жәҗз®ЎзҗҶгҖҒ AM/RM йҖҡдҝЎд»ҘеҸҠжұҮжҠҘзҠ¶жҖҒгҖӮ

    • Container пјҡ Container иҙҹиҙЈеҜ№иө„жәҗиҝӣиЎҢжҠҪиұЎпјҢеҢ…жӢ¬еҶ…еӯҳгҖҒ CPU гҖҒзЈҒзӣҳпјҢзҪ‘з»ңзӯүиө„жәҗгҖӮ

    2.3 Yarn жһ¶жһ„еҺҹзҗҶвҖ“дәӨдә’¶

    д»ҘеңЁ Yarn дёҠиҝҗиЎҢ MapReduce д»»еҠЎдёәдҫӢжқҘи®Іи§ЈдёӢ Yarn жһ¶жһ„зҡ„дәӨдә’еҺҹзҗҶпјҡ

    • йҰ–е…ҲпјҢз”ЁжҲ·зј–еҶҷ MapReduce д»Јз ҒеҗҺпјҢйҖҡиҝҮ Client з«ҜиҝӣиЎҢд»»еҠЎжҸҗдәӨ

    • ResourceManager еңЁжҺҘж”¶еҲ°е®ўжҲ·з«Ҝзҡ„иҜ·жұӮеҗҺпјҢдјҡеҲҶй…ҚдёҖдёӘ Container з”ЁжқҘеҗҜеҠЁ ApplicationMaster пјҢе№¶йҖҡзҹҘ NodeManager еңЁиҝҷдёӘ Container дёӢеҗҜеҠЁ ApplicationMaster гҖӮ

    • ApplicationMaster еҗҜеҠЁеҗҺпјҢеҗ‘ ResourceManager еҸ‘иө·жіЁеҶҢиҜ·жұӮгҖӮжҺҘзқҖ ApplicationMaster еҗ‘ ResourceManager з”іиҜ·иө„жәҗгҖӮж №жҚ®иҺ·еҸ–еҲ°зҡ„иө„жәҗпјҢе’Ңзӣёе…ізҡ„ NodeManager йҖҡдҝЎпјҢиҰҒжұӮе…¶еҗҜеҠЁзЁӢеәҸгҖӮ

    • дёҖдёӘжҲ–иҖ…еӨҡдёӘ NodeManager еҗҜеҠЁ Map/Reduce Task гҖӮ

    • NodeManager дёҚж–ӯжұҮжҠҘ Map/Reduce Task зҠ¶жҖҒе’Ңиҝӣеұ•з»ҷ ApplicationMaster гҖӮ

    • еҪ“жүҖжңү Map/Reduce Task йғҪе®ҢжҲҗж—¶пјҢ ApplicationMaster еҗ‘ ResourceManager жұҮжҠҘд»»еҠЎе®ҢжҲҗпјҢе№¶жіЁй”ҖиҮӘе·ұгҖӮ

    2.4 Flink on YarnвҖ“Per Job¶

    Flink on Yarn дёӯзҡ„ Per Job жЁЎејҸжҳҜжҢҮжҜҸж¬ЎжҸҗдәӨдёҖдёӘд»»еҠЎпјҢз„¶еҗҺд»»еҠЎиҝҗиЎҢе®ҢжҲҗд№ӢеҗҺиө„жәҗе°ұдјҡиў«йҮҠж”ҫгҖӮеңЁдәҶи§ЈдәҶ Yarn зҡ„еҺҹзҗҶд№ӢеҗҺпјҢ Per Job зҡ„жөҒзЁӢд№ҹе°ұжҜ”иҫғе®№жҳ“зҗҶи§ЈдәҶпјҢе…·дҪ“еҰӮдёӢпјҡ

    • йҰ–е…Ҳ Client жҸҗдәӨ Yarn App пјҢжҜ”еҰӮ JobGraph жҲ–иҖ… JARs гҖӮ

    • жҺҘдёӢжқҘ Yarn зҡ„ ResourceManager дјҡз”іиҜ·з¬¬дёҖдёӘ Container гҖӮиҝҷдёӘ Container йҖҡиҝҮ Application Master еҗҜеҠЁиҝӣзЁӢпјҢ Application Master йҮҢйқўиҝҗиЎҢзҡ„жҳҜ Flink зЁӢеәҸпјҢеҚі Flink-Yarn ResourceManager е’Ң JobManager гҖӮ

    • жңҖеҗҺ Flink-Yarn ResourceManager еҗ‘ Yarn ResourceManager з”іиҜ·иө„жәҗгҖӮеҪ“еҲҶй…ҚеҲ°иө„жәҗеҗҺпјҢеҗҜеҠЁ TaskManager гҖӮ TaskManager еҗҜеҠЁеҗҺеҗ‘ Flink-Yarn ResourceManager иҝӣиЎҢжіЁеҶҢпјҢжіЁеҶҢжҲҗеҠҹеҗҺ JobManager е°ұдјҡеҲҶй…Қе…·дҪ“зҡ„д»»еҠЎз»ҷ TaskManager ејҖе§Ӣжү§иЎҢгҖӮ

    2.5 Flink on YarnвҖ“Session¶

    еңЁ Per Job жЁЎејҸдёӯпјҢжү§иЎҢе®Ңд»»еҠЎеҗҺж•ҙдёӘиө„жәҗе°ұдјҡйҮҠж”ҫпјҢеҢ…жӢ¬ JobManager гҖҒ TaskManager йғҪе…ЁйғЁйҖҖеҮәгҖӮиҖҢ Session жЁЎејҸеҲҷдёҚдёҖж ·пјҢе®ғзҡ„ Dispatcher е’Ң ResourceManager жҳҜеҸҜд»ҘеӨҚз”Ёзҡ„гҖӮ Session жЁЎејҸдёӢпјҢеҪ“ Dispatcher еңЁж”¶еҲ°иҜ·жұӮд№ӢеҗҺпјҢдјҡеҗҜеҠЁ JobManager(A) пјҢи®© JobManager(A) жқҘе®ҢжҲҗеҗҜеҠЁ TaskManager пјҢжҺҘзқҖдјҡеҗҜеҠЁ JobManager(B) е’ҢеҜ№еә”зҡ„ TaskManager зҡ„иҝҗиЎҢгҖӮеҪ“ A гҖҒ B д»»еҠЎиҝҗиЎҢе®ҢжҲҗеҗҺпјҢиө„жәҗе№¶дёҚдјҡйҮҠж”ҫгҖӮ Session жЁЎејҸд№ҹз§°дёәеӨҡзәҝзЁӢжЁЎејҸпјҢе…¶зү№зӮ№жҳҜиө„жәҗдјҡдёҖзӣҙеӯҳеңЁдёҚдјҡйҮҠж”ҫпјҢеӨҡдёӘ JobManager е…ұдә«дёҖдёӘ Dispatcher пјҢиҖҢдё”иҝҳе…ұдә« Flink-YARN ResourceManager гҖӮ

    Session жЁЎејҸе’Ң Per Job жЁЎејҸзҡ„еә”з”ЁеңәжҷҜдёҚдёҖж ·гҖӮ Per Job жЁЎејҸжҜ”иҫғйҖӮеҗҲйӮЈз§ҚеҜ№еҗҜеҠЁж—¶й—ҙдёҚж•Ҹж„ҹпјҢиҝҗиЎҢж—¶й—ҙиҫғй•ҝзҡ„д»»еҠЎгҖӮ Seesion жЁЎејҸйҖӮеҗҲзҹӯж—¶й—ҙиҝҗиЎҢзҡ„д»»еҠЎпјҢдёҖиҲ¬жҳҜжү№еӨ„зҗҶд»»еҠЎгҖӮиӢҘз”Ё Per Job жЁЎејҸеҺ»иҝҗиЎҢзҹӯж—¶й—ҙзҡ„д»»еҠЎпјҢйӮЈе°ұйңҖиҰҒйў‘з№Ғзҡ„з”іиҜ·иө„жәҗпјҢиҝҗиЎҢз»“жқҹеҗҺпјҢиҝҳйңҖиҰҒиө„жәҗйҮҠж”ҫпјҢдёӢж¬ЎиҝҳйңҖеҶҚйҮҚж–°з”іиҜ·иө„жәҗжүҚиғҪиҝҗиЎҢгҖӮжҳҫз„¶пјҢиҝҷз§Қд»»еҠЎдјҡйў‘з№ҒеҗҜеҒңзҡ„жғ…еҶөдёҚйҖӮз”ЁдәҺ Per Job жЁЎејҸпјҢжӣҙйҖӮеҗҲз”Ё Session жЁЎејҸгҖӮ

    2.6 Yarn жЁЎејҸзү№зӮ№¶

    Yarn жЁЎејҸзҡ„дјҳзӮ№жңүпјҡ

    • иө„жәҗзҡ„з»ҹдёҖз®ЎзҗҶе’Ңи°ғеәҰгҖӮ Yarn йӣҶзҫӨдёӯжүҖжңүиҠӮзӮ№зҡ„иө„жәҗпјҲеҶ…еӯҳгҖҒ CPU гҖҒзЈҒзӣҳгҖҒзҪ‘з»ңзӯүпјүиў«жҠҪиұЎдёә Container гҖӮи®Ўз®—жЎҶжһ¶йңҖиҰҒиө„жәҗиҝӣиЎҢиҝҗз®—д»»еҠЎж—¶йңҖиҰҒеҗ‘ Resource Manager з”іиҜ· Container пјҢ YARN жҢүз…§зү№е®ҡзҡ„зӯ–з•ҘеҜ№иө„жәҗиҝӣиЎҢи°ғеәҰе’ҢиҝӣиЎҢ Container зҡ„еҲҶй…ҚгҖӮ Yarn жЁЎејҸиғҪйҖҡиҝҮеӨҡз§Қд»»еҠЎи°ғеәҰзӯ–з•ҘжқҘеҲ©з”ЁжҸҗй«ҳйӣҶзҫӨиө„жәҗеҲ©з”ЁзҺҮгҖӮдҫӢеҰӮ FIFO Scheduler гҖҒ Capacity Scheduler гҖҒ Fair Scheduler пјҢе№¶иғҪи®ҫзҪ®д»»еҠЎдјҳе…Ҳзә§гҖӮ

    • иө„жәҗйҡ”зҰ»пјҡ Yarn дҪҝз”ЁдәҶиҪ»йҮҸзә§иө„жәҗйҡ”зҰ»жңәеҲ¶ Cgroups иҝӣиЎҢиө„жәҗйҡ”зҰ»д»ҘйҒҝе…Қзӣёдә’е№Іжү°пјҢдёҖж—Ұ Container дҪҝз”Ёзҡ„иө„жәҗйҮҸи¶…иҝҮдәӢе…Ҳе®ҡд№үзҡ„дёҠйҷҗеҖјпјҢе°ұе°Ҷе…¶жқҖжӯ»гҖӮ

    • иҮӘеҠЁ failover еӨ„зҗҶгҖӮдҫӢеҰӮ Yarn NodeManager зӣ‘жҺ§гҖҒ Yarn ApplicationManager ејӮеёёжҒўеӨҚгҖӮ

    Yarn жЁЎејҸиҷҪз„¶жңүдёҚе°‘дјҳзӮ№пјҢдҪҶжҳҜд№ҹжңүиҜёеӨҡзјәзӮ№пјҢдҫӢеҰӮиҝҗз»ҙйғЁзҪІжҲҗжң¬иҫғй«ҳпјҢзҒөжҙ»жҖ§дёҚеӨҹгҖӮ

    2.7 Flink on Yarn е®һи·ө¶

    е…ідәҺ Flink on Yarn зҡ„е®һи·өеңЁжң¬з«ҷдёҠйқўжңүеҫҲеӨҡиҜҫзЁӢпјҢдҫӢеҰӮпјҡгҖҠ Flink е®үиЈ…йғЁзҪІгҖҒзҺҜеўғй…ҚзҪ®еҸҠиҝҗиЎҢеә”з”ЁзЁӢеәҸгҖӢе’ҢгҖҠе®ўжҲ·з«Ҝж“ҚдҪңгҖӢйғҪжҳҜеҹәдәҺ Yarn иҝӣиЎҢи®Іи§Јзҡ„пјҢиҝҷйҮҢе°ұдёҚеҶҚиөҳиҝ°гҖӮ

    3 Flink on Kubernetes еҺҹзҗҶеү–жһҗ¶

    Kubernetes жҳҜ Google ејҖжәҗзҡ„е®№еҷЁйӣҶзҫӨз®ЎзҗҶзі»з»ҹпјҢе…¶жҸҗдҫӣеә”з”ЁйғЁзҪІгҖҒз»ҙжҠӨгҖҒжү©еұ•жңәеҲ¶зӯүеҠҹиғҪпјҢеҲ©з”Ё Kubernetes иғҪж–№дҫҝең°з®ЎзҗҶи·ЁжңәеҷЁиҝҗиЎҢе®№еҷЁеҢ–зҡ„еә”з”ЁгҖӮ Kubernetes е’Ң Yarn зӣёжҜ”пјҢзӣёеҪ“дәҺдёӢдёҖд»Јзҡ„иө„жәҗз®ЎзҗҶзі»з»ҹпјҢдҪҶжҳҜе®ғзҡ„иғҪеҠӣиҝңиҝңдёҚжӯўиҝҷдәӣгҖӮ

    3.1 KubernetesвҖ“еҹәжң¬жҰӮеҝө¶

    Kubernetes(k8s) дёӯзҡ„ Master иҠӮзӮ№пјҢиҙҹиҙЈз®ЎзҗҶж•ҙдёӘйӣҶзҫӨпјҢеҗ«жңүдёҖдёӘйӣҶзҫӨзҡ„иө„жәҗж•°жҚ®и®ҝй—®е…ҘеҸЈпјҢиҝҳеҢ…еҗ«дёҖдёӘ Etcd й«ҳеҸҜз”Ёй”®еҖјеӯҳеӮЁжңҚеҠЎгҖӮ Master дёӯиҝҗиЎҢзқҖ API Server пјҢ Controller Manager еҸҠ Scheduler жңҚеҠЎгҖӮ

    Node дёәйӣҶзҫӨзҡ„дёҖдёӘж“ҚдҪңеҚ•е…ғпјҢжҳҜ Pod иҝҗиЎҢзҡ„е®ҝдё»жңәгҖӮ Node иҠӮзӮ№йҮҢеҢ…еҗ«дёҖдёӘ agent иҝӣзЁӢпјҢиғҪеӨҹз»ҙжҠӨе’Ңз®ЎзҗҶиҜҘ Node дёҠзҡ„жүҖжңүе®№еҷЁзҡ„еҲӣе»әгҖҒеҗҜеҒңзӯүгҖӮ Node иҝҳеҗ«жңүдёҖдёӘжңҚеҠЎз«Ҝ kube-proxy пјҢз”ЁдәҺжңҚеҠЎеҸ‘зҺ°гҖҒеҸҚеҗ‘д»ЈзҗҶе’ҢиҙҹиҪҪеқҮиЎЎгҖӮ Node еә•еұӮеҗ«жңү docker engine пјҢ docker еј•ж“Һдё»иҰҒиҙҹиҙЈжң¬жңәе®№еҷЁзҡ„еҲӣе»әе’Ңз®ЎзҗҶе·ҘдҪңгҖӮ

    Pod иҝҗиЎҢдәҺ Node иҠӮзӮ№дёҠпјҢжҳҜиӢҘе№Ізӣёе…іе®№еҷЁзҡ„з»„еҗҲгҖӮеңЁ K8s йҮҢйқў Pod жҳҜеҲӣе»әгҖҒи°ғеәҰе’Ңз®ЎзҗҶзҡ„жңҖе°ҸеҚ•дҪҚгҖӮ

    3.2 KubernetesвҖ“жһ¶жһ„еӣҫ¶

    Kubernetes зҡ„жһ¶жһ„еҰӮеӣҫжүҖзӨәпјҢд»ҺиҝҷдёӘеӣҫйҮҢйқўиғҪзңӢеҮә Kubernetes зҡ„ж•ҙдёӘиҝҗиЎҢиҝҮзЁӢгҖӮ

    • API Server зӣёеҪ“дәҺз”ЁжҲ·зҡ„дёҖдёӘиҜ·жұӮе…ҘеҸЈпјҢз”ЁжҲ·еҸҜд»ҘжҸҗдәӨе‘Ҫд»Өз»ҷ Etcd пјҢиҝҷж—¶дјҡе°ҶиҝҷдәӣиҜ·жұӮеӯҳеӮЁеҲ° Etcd йҮҢйқўеҺ»гҖӮ

    • Etcd жҳҜдёҖдёӘй”®еҖјеӯҳеӮЁпјҢиҙҹиҙЈе°Ҷд»»еҠЎеҲҶй…Қз»ҷе…·дҪ“зҡ„жңәеҷЁпјҢеңЁжҜҸдёӘиҠӮзӮ№дёҠзҡ„ Kubelet дјҡжүҫеҲ°еҜ№еә”зҡ„ container еңЁжң¬жңәдёҠиҝҗиЎҢгҖӮ

    • з”ЁжҲ·еҸҜд»ҘжҸҗдәӨдёҖдёӘ Replication Controller иө„жәҗжҸҸиҝ°пјҢ Replication Controller дјҡзӣ‘и§ҶйӣҶзҫӨдёӯзҡ„е®№еҷЁе№¶дҝқжҢҒж•°йҮҸпјӣз”ЁжҲ·д№ҹеҸҜд»ҘжҸҗдәӨ service жҸҸиҝ°ж–Үд»¶пјҢе№¶з”ұ kube proxy иҙҹиҙЈе…·дҪ“е·ҘдҪңзҡ„жөҒйҮҸиҪ¬еҸ‘гҖӮ

    3.3 KubernetesвҖ“ж ёеҝғжҰӮеҝө¶

    Kubernetes дёӯжҜ”иҫғйҮҚиҰҒзҡ„жҰӮеҝөжңүпјҡ

    • Replication Controller (RC) з”ЁжқҘз®ЎзҗҶ Pod зҡ„еүҜжң¬гҖӮ RC зЎ®дҝқд»»дҪ•ж—¶еҖҷ Kubernetes йӣҶзҫӨдёӯжңүжҢҮе®ҡж•°йҮҸзҡ„ pod еүҜжң¬пјҲ replicas пјүеңЁиҝҗиЎҢпјҢеҰӮжһңе°‘дәҺжҢҮе®ҡж•°йҮҸзҡ„ pod еүҜжң¬пјҢ RC дјҡеҗҜеҠЁж–°зҡ„ Container пјҢеҸҚд№ӢдјҡжқҖжӯ»еӨҡдҪҷзҡ„д»ҘдҝқиҜҒж•°йҮҸдёҚеҸҳгҖӮ

    • Service жҸҗдҫӣдәҶдёҖдёӘз»ҹдёҖзҡ„жңҚеҠЎи®ҝй—®е…ҘеҸЈд»ҘеҸҠжңҚеҠЎд»ЈзҗҶе’ҢеҸ‘зҺ°жңәеҲ¶

    • Persistent Volume(PV) е’Ң Persistent Volume Claim(PVC) з”ЁдәҺж•°жҚ®зҡ„жҢҒд№…еҢ–еӯҳеӮЁгҖӮ

    • ConfigMap жҳҜжҢҮеӯҳеӮЁз”ЁжҲ·зЁӢеәҸзҡ„й…ҚзҪ®ж–Үд»¶пјҢе…¶еҗҺз«ҜеӯҳеӮЁжҳҜеҹәдәҺ Etcd гҖӮ

    3.4 Flink on KubernetesвҖ“жһ¶жһ„¶

    Flink on Kubernetes зҡ„жһ¶жһ„еҰӮеӣҫжүҖзӨәпјҢ Flink д»»еҠЎеңЁ Kubernetes дёҠиҝҗиЎҢзҡ„жӯҘйӘӨжңүпјҡ

    • йҰ–е…ҲеҫҖ Kubernetes йӣҶзҫӨжҸҗдәӨдәҶиө„жәҗжҸҸиҝ°ж–Үд»¶еҗҺпјҢдјҡеҗҜеҠЁ Master е’Ң Worker зҡ„ container гҖӮ

    • Master Container дёӯдјҡеҗҜеҠЁ Flink Master Process пјҢеҢ…еҗ« Flink-Container ResourceManager гҖҒ JobManager е’Ң Program Runner гҖӮ

    • Worker Container дјҡеҗҜеҠЁ TaskManager пјҢе№¶еҗ‘иҙҹиҙЈиө„жәҗз®ЎзҗҶзҡ„ ResourceManager иҝӣиЎҢжіЁеҶҢпјҢжіЁеҶҢе®ҢжҲҗд№ӢеҗҺпјҢз”ұ JobManager е°Ҷе…·дҪ“зҡ„д»»еҠЎеҲҶз»ҷ Container пјҢеҶҚз”ұ Container еҺ»жү§иЎҢгҖӮ

    • йңҖиҰҒиҜҙжҳҺзҡ„жҳҜпјҢеңЁ Flink йҮҢзҡ„ Master е’Ң Worker йғҪжҳҜдёҖдёӘй•ңеғҸпјҢеҸӘжҳҜи„ҡжң¬зҡ„е‘Ҫд»ӨдёҚдёҖж ·пјҢйҖҡиҝҮеҸӮж•°жқҘйҖүжӢ©еҗҜеҠЁ master иҝҳжҳҜеҗҜеҠЁ Worker гҖӮ

    3.5 Flink on KubernetesвҖ“JobManager¶

    JobManager зҡ„жү§иЎҢиҝҮзЁӢеҲҶдёәдёӨжӯҘпјҡ

    • йҰ–е…ҲпјҢ JobManager йҖҡиҝҮ Deployment иҝӣиЎҢжҸҸиҝ°пјҢдҝқиҜҒ 1 дёӘеүҜжң¬зҡ„ Container иҝҗиЎҢ JobManager пјҢеҸҜд»Ҙе®ҡд№үдёҖдёӘж ҮзӯҫпјҢдҫӢеҰӮ flink-jobmanager гҖӮ

    • е…¶ж¬ЎпјҢиҝҳйңҖиҰҒе®ҡд№үдёҖдёӘ JobManager Service пјҢйҖҡиҝҮ service name е’Ң port жҡҙйңІ JobManager жңҚеҠЎпјҢйҖҡиҝҮж ҮзӯҫйҖүжӢ©еҜ№еә”зҡ„ pods гҖӮ

    3.6 Flink on KubernetesвҖ“TaskManager¶

    TaskManager д№ҹжҳҜйҖҡиҝҮ Deployment жқҘиҝӣиЎҢжҸҸиҝ°пјҢдҝқиҜҒ n дёӘеүҜжң¬зҡ„ Container иҝҗиЎҢ TaskManager пјҢеҗҢж—¶д№ҹйңҖиҰҒе®ҡд№үдёҖдёӘж ҮзӯҫпјҢдҫӢеҰӮ flink-taskmanager гҖӮ

    еҜ№дәҺ JobManager е’Ң TaskManager иҝҗиЎҢиҝҮзЁӢдёӯйңҖиҰҒзҡ„дёҖдәӣй…ҚзҪ®ж–Үд»¶пјҢеҰӮпјҡ flink-conf.yaml гҖҒ hdfs-site.xml гҖҒ core-site.xml пјҢеҸҜд»ҘйҖҡиҝҮе°Ҷе®ғд»¬е®ҡд№үдёә ConfigMap жқҘе®һзҺ°й…ҚзҪ®зҡ„дј йҖ’е’ҢиҜ»еҸ–гҖӮ

    3.7 Flink on KubernetesвҖ“дәӨдә’¶

    ж•ҙдёӘдәӨдә’зҡ„жөҒзЁӢжҜ”иҫғз®ҖеҚ•пјҢз”ЁжҲ·еҫҖ Kubernetes йӣҶзҫӨжҸҗдәӨе®ҡд№үеҘҪзҡ„иө„жәҗжҸҸиҝ°ж–Үд»¶еҚіеҸҜпјҢдҫӢеҰӮ deployment гҖҒ configmap гҖҒ service зӯүжҸҸиҝ°гҖӮеҗҺз»ӯзҡ„дәӢжғ…е°ұдәӨз»ҷ Kubernetes йӣҶзҫӨиҮӘеҠЁе®ҢжҲҗгҖӮ Kubernetes йӣҶзҫӨдјҡжҢүз…§е®ҡд№үеҘҪзҡ„жҸҸиҝ°жқҘеҗҜеҠЁ pod пјҢиҝҗиЎҢз”ЁжҲ·зЁӢеәҸгҖӮеҗ„дёӘз»„д»¶зҡ„е…·дҪ“е·ҘдҪңеҰӮдёӢпјҡ

    • Service пјҡйҖҡиҝҮж ҮзӯҫпјҲ label selector пјүжүҫеҲ° job manager зҡ„ pod жҡҙйңІжңҚеҠЎгҖӮ

    • Deployment пјҡдҝқиҜҒ n дёӘеүҜжң¬зҡ„ container иҝҗиЎҢ JM/TM пјҢеә”з”ЁеҚҮзә§зӯ–з•ҘгҖӮ

    • ConfigMap пјҡеңЁжҜҸдёӘ pod дёҠйҖҡиҝҮжҢӮиҪҪ /etc/flink зӣ®еҪ•пјҢеҢ…еҗ« flink-conf.yaml еҶ…е®№гҖӮ

    еҸіеӣҫдёә taskmanager-deployment.yaml й…ҚзҪ®пјҢ taskmanager-deployment.yaml й…ҚзҪ®дёҺ jobmanager-deployment.yaml зӣёдјјпјҢдҪҶ taskmanager-deployment.yaml зҡ„еүҜжң¬ж•°жҳҜ 2 дёӘгҖӮ

    жҺҘдёӢжқҘжҳҜ jobmanager-service.yaml зҡ„й…ҚзҪ®пјҢ jobmanager-service.yaml зҡ„иө„жәҗзұ»еһӢдёә Service пјҢеңЁ Service дёӯзҡ„й…ҚзҪ®зӣёеҜ№е°‘дёҖдәӣпјҢ spec дёӯй…ҚзҪ®йңҖиҰҒжҡҙйңІзҡ„жңҚеҠЎз«ҜеҸЈзҡ„ port пјҢеңЁ selector дёӯпјҢйҖҡиҝҮж ҮзӯҫйҖүеҸ– jobmanager зҡ„ pod гҖӮ
