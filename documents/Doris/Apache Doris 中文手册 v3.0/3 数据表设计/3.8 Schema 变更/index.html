    • Transaction пјҡдәӢеҠЎгҖӮжҜҸдёҖдёӘеҜје…Ҙд»»еҠЎйғҪеҜ№еә”дёҖдёӘдәӢеҠЎпјҢжҜҸдёӘдәӢеҠЎжңүдёҖдёӘе”ҜдёҖйҖ’еўһзҡ„ Transaction ID гҖӮ

    2 еҺҹзҗҶд»Ӣз»Қ¶

    2.1 жҰӮи§Ҳ¶

    Schema Change зҡ„е®һзҺ°еҲҶдёәдёӨдёӘеӨ§зұ»пјҡиҪ»йҮҸзә§ Schema Change е’ҢйҮҚйҮҸзә§ Schema Change гҖӮ

    • иҪ»йҮҸзә§ Schema Change е®ҢжҲҗйҖҹеәҰеҫҲеҝ«пјҢеҸӘдјҡеҗҢжӯҘең°дҝ®ж”№ FE зҡ„е…ғж•°жҚ®пјҢдёҖиҲ¬еңЁз§’зә§еҲ«е®ҢжҲҗгҖӮеўһеҠ жҲ–еҲ йҷӨ value еҲ—гҖҒжӣҙж”№еҲ—еҗҚгҖҒеўһеҠ йҷӨ DUP KEY еҲ—е’Ң UNIQUE KEY еҲ—д»ҘеӨ–зҡ„ VARCHAR еҲ—зҡ„й•ҝеәҰпјҢйғҪдјҡдҪҝз”ЁиҪ»йҮҸзә§ Schema Change зҡ„йҖ»иҫ‘гҖӮ

    • йҮҚйҮҸзә§ Schema Change йңҖиҰҒдҫқиө– BE иҝӣиЎҢж•°жҚ®ж–Үд»¶зҡ„иҪ¬жҚўгҖӮе…·дҪ“е®һзҺ°ж–№ејҸеҰӮдёӢпјҡ

      Schema changeе®һзҺ° дё»иҰҒйҖ»иҫ‘ дҪҝз”ЁеңәжҷҜ
      Direct Schema Change еҜ№ж•°жҚ®ж–Үд»¶иҝӣиЎҢж•ҙдҪ“йҮҚеҶҷпјҢдҪҶжҳҜдёҚдјҡж¶үеҸҠйҮҚжҺ’еәҸ жӣҙж”№valueеҲ—зҡ„ж•°жҚ®зұ»еһӢ
      Sort Schema Change еҜ№ж•°жҚ®ж–Үд»¶иҝӣиЎҢж•ҙдҪ“йҮҚеҶҷпјҢе№¶иҝӣиЎҢйҮҚжҺ’еәҸ жӣҙж”№keyеҲ—зҡ„ж•°жҚ®зұ»еһӢ
      Hard Linked Schema Change еҜ№ж•°жҚ®ж–Үд»¶иҝӣиЎҢйҮҚж–°й“ҫжҺҘпјҢдёҚйңҖиҰҒзӣҙжҺҘдҝ®ж”№ж•°жҚ®ж–Үд»¶ еңЁеҲ—зҡ„еҸҳжӣҙдёӯиў«иҪ»йҮҸзә§Schema ChangeеҸ–д»Ј
