    Delete ж“ҚдҪңиҜӯеҸҘйҖҡиҝҮ MySQL еҚҸи®®пјҢеҜ№жҢҮе®ҡзҡ„ table жҲ–иҖ… partition дёӯзҡ„ж•°жҚ®иҝӣиЎҢжҢүжқЎд»¶еҲ йҷӨгҖӮ Delete еҲ йҷӨж“ҚдҪңдёҚеҗҢдәҺеҹәдәҺеҜје…Ҙзҡ„жү№йҮҸеҲ йҷӨпјҢе®ғзұ»дјј Insert into иҜӯеҸҘпјҢжҳҜдёҖдёӘеҗҢжӯҘиҝҮзЁӢгҖӮжүҖжңүзҡ„ Delete ж“ҚдҪңеңЁ Doris дёӯжҳҜдёҖдёӘзӢ¬з«Ӣзҡ„еҜје…ҘдҪңдёҡпјҢдёҖиҲ¬ Delete иҜӯеҸҘйңҖиҰҒжҢҮе®ҡиЎЁе’ҢеҲҶеҢәд»ҘеҸҠеҲ йҷӨзҡ„жқЎд»¶жқҘзӯӣйҖүиҰҒеҲ йҷӨзҡ„ж•°жҚ®пјҢе№¶е°ҶдјҡеҗҢж—¶еҲ йҷӨ base иЎЁе’Ң rollup иЎЁзҡ„ж•°жҚ®гҖӮ

    Delete ж“ҚдҪңзҡ„иҜӯжі•иҜҰи§Ғ DELETE иҜӯжі•гҖӮдёҚеҗҢдәҺ Insert into е‘Ҫд»ӨпјҢ delete дёҚиғҪжүӢеҠЁжҢҮе®ҡ label пјҢжңүе…і label зҡ„жҰӮеҝөеҸҜд»ҘжҹҘзңӢ Insert Into ж–ҮжЎЈгҖӮ

    1 йҖҡиҝҮжҢҮе®ҡиҝҮж»Өи°“иҜҚжқҘеҲ йҷӨ¶

    1.1 еҝ…йЎ»зҡ„еҸӮж•°¶

    • table_name пјҡжҢҮе®ҡйңҖиҰҒеҲ йҷӨж•°жҚ®зҡ„иЎЁ

    • column_name пјҡеұһдәҺ table_name зҡ„еҲ—

    • op пјҡйҖ»иҫ‘жҜ”иҫғж“ҚдҪңз¬ҰпјҢеҸҜйҖүзұ»еһӢеҢ…жӢ¬пјҡ = пјҢ > пјҢ < пјҢ >= пјҢ <= пјҢ != пјҢ in пјҢ not in

    • value | value_list пјҡеҒҡйҖ»иҫ‘жҜ”иҫғзҡ„еҖјжҲ–еҖјеҲ—иЎЁ

    1.2 еҸҜйҖүзҡ„еҸӮж•°¶

    • PARTITION partition_name | PARTITIONS (partition_name [, partition_name]) пјҡжҢҮе®ҡжү§иЎҢеҲ йҷӨж•°жҚ®зҡ„еҲҶеҢәеҗҚпјҢеҰӮжһңиЎЁдёҚеӯҳеңЁжӯӨеҲҶеҢәпјҢеҲҷжҠҘй”ҷ

    • table_alias пјҡиЎЁзҡ„еҲ«еҗҚ

    1.3 жіЁж„ҸдәӢйЎ№¶

    • дҪҝз”ЁиЎЁжЁЎеһӢ Aggregate ж—¶пјҢеҸӘиғҪжҢҮе®ҡ Key еҲ—дёҠзҡ„жқЎд»¶гҖӮ

    • еҪ“йҖүе®ҡзҡ„ Key еҲ—дёҚеӯҳеңЁдәҺжҹҗдёӘ Rollup дёӯж—¶пјҢж— жі•иҝӣиЎҢ Delete гҖӮ

    • жқЎд»¶д№Ӣй—ҙеҸӘиғҪжҳҜвҖңдёҺвҖқзҡ„е…ізі»гҖӮиӢҘеёҢжңӣиҫҫжҲҗвҖңжҲ–вҖқзҡ„е…ізі»пјҢйңҖиҰҒе°ҶжқЎд»¶еҲҶеҶҷеңЁдёӨдёӘ DELETE иҜӯеҸҘдёӯпјӣ

    • еҰӮжһңдёәеҲҶеҢәиЎЁпјҢйңҖиҰҒжҢҮе®ҡеҲҶеҢәпјҢеҰӮжһңдёҚжҢҮе®ҡпјҢ Doris дјҡд»ҺжқЎд»¶дёӯжҺЁж–ӯеҮәеҲҶеҢәгҖӮдёӨз§Қжғ…еҶөдёӢпјҢ Doris ж— жі•д»ҺжқЎд»¶дёӯжҺЁж–ӯеҮәеҲҶеҢәпјҡ

    • жқЎд»¶дёӯдёҚеҢ…еҗ«еҲҶеҢәеҲ—пјӣ

    • еҲҶеҢәеҲ—зҡ„ op дёә not in гҖӮеҪ“еҲҶеҢәиЎЁжңӘжҢҮе®ҡеҲҶеҢәпјҢжҲ–иҖ…ж— жі•д»ҺжқЎд»¶дёӯжҺЁж–ӯеҲҶеҢәзҡ„ж—¶еҖҷпјҢйңҖиҰҒи®ҫзҪ®дјҡиҜқеҸҳйҮҸ delete_without_partition дёә true пјҢжӯӨж—¶ Delete дјҡеә”з”ЁеҲ°жүҖжңүеҲҶеҢәгҖӮ

    • иҜҘиҜӯеҸҘеҸҜиғҪдјҡйҷҚдҪҺжү§иЎҢеҗҺдёҖж®өж—¶й—ҙеҶ…зҡ„жҹҘиҜўж•ҲзҺҮгҖӮеҪұе“ҚзЁӢеәҰеҸ–еҶідәҺиҜӯеҸҘдёӯжҢҮе®ҡзҡ„еҲ йҷӨжқЎд»¶зҡ„ж•°йҮҸгҖӮжҢҮе®ҡзҡ„жқЎд»¶и¶ҠеӨҡпјҢеҪұе“Қи¶ҠеӨ§гҖӮ
