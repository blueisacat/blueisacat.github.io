    еҜ№дәҺеӯҳеӮЁд»ӢиҙЁдёәзЈҒзӣҳжҲ– SSD зҡ„ж•°жҚ®еә“пјҢй•ҝжңҹд»ҘжқҘдё»жөҒдҪҝз”Ё B+ ж ‘иҝҷз§Қзҙўеј•з»“жһ„жқҘе®һзҺ°еҝ«йҖҹж•°жҚ®жҹҘжүҫгҖӮеҪ“ж•°жҚ®йҮҸдёҚеӨӘеӨ§ж—¶пјҢ B+ ж ‘иҜ»еҶҷжҖ§иғҪиЎЁзҺ°йқһеёёеҘҪгҖӮдҪҶжҳҜеңЁжө·йҮҸж•°жҚ®жғ…еҶөдёӢпјҢ B+ ж ‘и¶ҠжқҘи¶Ҡй«ҳпјҢз”ұдәҺ B+ ж ‘жӣҙж–°е’ҢеҲ йҷӨж•°жҚ®ж—¶йңҖиҰҒжІҝзқҖ B+ ж ‘йҖҗеұӮиҝӣиЎҢйЎөеҲҶиЈӮе’ҢйЎөеҗҲе№¶пјҢдёҘйҮҚеҪұе“Қж•°жҚ®еҶҷе…ҘжҖ§иғҪгҖӮдёәдәҶеә”еҜ№иҝҷз§Қжғ…еҶөпјҢ google еңЁи®әж–ҮгҖҠ Bigtable: A Distributed Storage System for Structured Data гҖӢдёӯд»Ӣз»ҚдәҶдёҖз§Қж–°зҡ„ж•°жҚ®з»„з»Үз»“жһ„ LSM Tree(Log-Structured Merge Tree) пјҢйҡҸеҗҺпјҢ Bigtable дё»иҰҒдҪңиҖ… Jeffrey Dean е’Ң Sanjay Ghemawat ејҖжәҗдәҶдёҖж¬ҫеҹәдәҺ LSM Tree е®һзҺ°зҡ„ж•°жҚ®еә“ LevelDB пјҢи®©еӨ§е®¶еҜ№ LSM Tree зҡ„жҖқжғіе’Ңе®һзҺ°зҗҶи§Јеҫ—жӣҙдёәйҖҸеҪ»гҖҒж·ұе…ҘгҖӮеҪ“еүҚпјҢжҜ”иҫғжөҒиЎҢзҡ„ NoSQL ж•°жҚ®еә“пјҢеҰӮ Cassandra гҖҒ RocksDB гҖҒ HBase гҖҒ LevelDB зӯүпјҢ newSQL ж•°жҚ®еә“пјҢеҰӮ TiDB пјҢеқҮжҳҜдҪҝз”Ё LSM Tree жқҘз»„з»ҮзЈҒзӣҳж•°жҚ®зҡ„гҖӮз”ҡиҮіеғҸ SQLite иҝҷз§Қдј з»ҹзҡ„е…ізі»еһӢж•°жҚ®еә“е’Ң MongoDB иҝҷз§Қдј з»ҹзҡ„ж–ҮжЎЈеһӢж•°жҚ®еә“йғҪжҸҗдҫӣдәҶеҹәдәҺ LSM Tree зҡ„еӯҳеӮЁеј•ж“ҺдҪңдёәеҸҜйҖүзҡ„еӯҳеӮЁеј•ж“ҺгҖӮ

    2 еҹәжң¬еҺҹзҗҶз®Җиҝ°¶

    LSM Tree зҡ„е…Ёз§°дёә Log-Structured Merge Tree пјҢжҳҜдёҖдёӘеҲҶеұӮгҖҒжңүеәҸгҖҒй’ҲеҜ№еқ—еӯҳеӮЁи®ҫеӨҮпјҲжңәжў°зЎ¬зӣҳе’Ң SSD пјүзү№зӮ№иҖҢи®ҫи®Ўзҡ„ж•°жҚ®еӯҳеӮЁз»“жһ„гҖӮе®ғзҡ„ж ёеҝғзҗҶи®әеҹәзЎҖиҝҳжҳҜзЈҒзӣҳзҡ„йЎәеәҸеҶҷйҖҹеәҰжҜ”йҡҸжңәеҶҷйҖҹеәҰеҝ«йқһеёёеӨҡпјҢеҚідҪҝжҳҜ SSD пјҢз”ұдәҺеқ—ж“ҰйҷӨе’Ңеһғеңҫеӣһж”¶зҡ„еҪұе“ҚпјҢйЎәеәҸеҶҷйҖҹеәҰиҝҳжҳҜжҜ”йҡҸжңәеҶҷйҖҹеәҰеҝ«еҫҲеӨҡгҖӮ

    LSM Tree е°ҶеӯҳеӮЁж•°жҚ®еҲҮеҲҶдёәдёҖзі»еҲ—зҡ„ SSTable пјҲ Sorted String Table пјүпјҢдёҖдёӘ SSTable еҶ…зҡ„ж•°жҚ®жҳҜжңүеәҸзҡ„д»»ж„Ҹеӯ—иҠӮз»„пјҲеҚі arbitrary byte string пјҢе№¶дёҚжҳҜжҢҮзј–зЁӢиҜӯиЁҖдёӯзҡ„ String еӯ—з¬ҰдёІпјүпјҢиҖҢдё”пјҢ SSTable дёҖдҪҶеҶҷе…ҘзЈҒзӣҳдёӯпјҢе°ұеғҸж—Ҙеҝ—дёҖж ·дёҚиғҪеҶҚдҝ®ж”№пјҲиҝҷе°ұжҳҜ Log-Structured Merge Tree еҗҚеӯ—дёӯ Log-Structured дёҖиҜҚзҡ„з”ұжқҘпјүгҖӮеҪ“иҰҒдҝ®ж”№зҺ°жңүж•°жҚ®ж—¶пјҢ LSM Tree е№¶дёҚзӣҙжҺҘдҝ®ж”№ж—§ж•°жҚ®пјҢиҖҢжҳҜзӣҙжҺҘе°Ҷж–°ж•°жҚ®еҶҷе…Ҙж–°зҡ„ SSTable дёӯгҖӮеҗҢж ·зҡ„пјҢеҲ йҷӨж•°жҚ®ж—¶пјҢ LSM Tree д№ҹдёҚзӣҙжҺҘеҲ йҷӨж—§ж•°жҚ®пјҢиҖҢжҳҜеҶҷдёҖдёӘзӣёеә”ж•°жҚ®зҡ„еҲ йҷӨж Үи®°зҡ„и®°еҪ•еҲ°дёҖдёӘж–°зҡ„ SSTable дёӯгҖӮиҝҷж ·дёҖжқҘпјҢ LSM Tree еҶҷж•°жҚ®ж—¶еҜ№зЈҒзӣҳзҡ„ж“ҚдҪңйғҪжҳҜйЎәеәҸеқ—еҶҷе…Ҙж“ҚдҪңпјҢиҖҢжІЎжңүйҡҸжңәеҶҷж“ҚдҪңгҖӮ

    LSM Tree иҝҷз§ҚзӢ¬зү№зҡ„еҶҷе…Ҙж–№ејҸпјҢеҜјиҮҙеңЁжҹҘжүҫж•°жҚ®ж—¶пјҢ LSM Tree е°ұдёҚиғҪеғҸ B+ ж ‘йӮЈж ·еңЁдёҖдёӘз»ҹдёҖзҡ„зҙўеј•иЎЁдёӯиҝӣиЎҢжҹҘжүҫпјҢиҖҢжҳҜд»ҺжңҖж–°зҡ„ SSTable еҲ°иҖҒзҡ„ SSTable дҫқж¬ЎиҝӣиЎҢжҹҘжүҫгҖӮеҰӮжһңеңЁж–° SSTable дёӯжүҫеҲ°дәҶйңҖжҹҘжүҫзҡ„ж•°жҚ®жҲ–зӣёеә”зҡ„еҲ йҷӨж Үи®°пјҢеҲҷзӣҙжҺҘиҝ”еӣһжҹҘжүҫз»“жһңпјӣеҰӮжһңжІЎжңүжүҫеҲ°пјҢеҶҚеҲ°иҖҒзҡ„ SSTable дёӯиҝӣиЎҢжҹҘжүҫпјҢзӣҙеҲ°жңҖиҖҒзҡ„ SSTable жҹҘжүҫе®ҢгҖӮдёәдәҶжҸҗй«ҳжҹҘжүҫж•ҲзҺҮпјҢ LSM Tree еҜ№ SSTable иҝӣиЎҢеҲҶеұӮгҖҒжңүеәҸз»„з»ҮпјҢд№ҹе°ұжҳҜиҜҙжҠҠ SSTable з»„з»ҮжҲҗеӨҡеұӮпјҢеҗҢдёҖеұӮеҸҜд»ҘжңүеӨҡдёӘ SSTable пјҢеҗҢдёҖдёӘж•°жҚ®еңЁеҗҢдёҖеұӮзҡ„еӨҡдёӘ SSTable дёӯеҸҜд»ҘдёҚйҮҚеӨҚпјҢиҖҢдё”ж•°жҚ®еҸҜд»ҘеҒҡеҲ°еңЁеҗҢдёҖеұӮдёӯжҳҜжңүеәҸзҡ„пјҢеҚіжҜҸдёҖдёӘ SSTable еҶ…зҡ„ж•°жҚ®жҳҜжңүеәҸзҡ„пјҢеүҚдёҖдёӘ SSTable зҡ„жңҖеӨ§ж•°жҚ®еҖје°ҸдәҺеҗҺдёҖдёӘ SSTable зҡ„жңҖе°Ҹж•°жҚ®еҖјпјҲе®һйҷ…жғ…еҶөжҜ”иҝҷдёӘеӨҚжқӮпјҢеҗҺйқўдјҡд»Ӣз»ҚпјүгҖӮиҝҷж ·еҸҜд»ҘеҠ еҝ«еңЁеҗҢдёҖеұӮ SSTable дёӯзҡ„ж•°жҚ®жҹҘиҜўйҖҹеәҰгҖӮеҗҢж—¶пјҢ LSM Tree дјҡе°ҶеӨҡдёӘ SSTable еҗҲе№¶пјҲ Compact пјүдёәдёҖдёӘж–°зҡ„ SSTable пјҢиҝҷж ·еҸҜд»ҘеҮҸе°‘ SSTable зҡ„ж•°йҮҸпјҢеҗҢж—¶жҠҠдҝ®ж”№еүҚзҡ„ж•°жҚ®жҲ–еҲ йҷӨзҡ„ж•°жҚ®зңҹжӯЈд»Һ SSTable дёӯеҲ йҷӨпјҢеҮҸе°ҸдәҶ SSTable зҡ„еӨ§е°ҸпјҲиҝҷе°ұжҳҜ Log-Structured Merge Tree еҗҚеӯ—дёӯ Merge дёҖиҜҚзҡ„з”ұжқҘпјүпјҢеҜ№жҸҗй«ҳжҹҘжүҫжҖ§иғҪжһҒе…¶йҮҚиҰҒпјҲ SSTable еҗҲе№¶пјҲ Compact пјүиҝҮзЁӢеҜ№ LSM Tree жҹҘжүҫеҰӮжӯӨйҮҚиҰҒпјҢд»ҘиҮідәҺжҠҠе®ғдҪңдёәеҗҚеӯ—зҡ„дёҖйғЁеҲҶпјүгҖӮ

    Tip

    An SSTable provides a persistent, ordered immutable map from keys to values, where both keys and values are arbitrary byte strings. Operations are provided to look up the value associated with a specified key, and to iterate over all key/value pairs in a specified key range. Internally, each SSTable contains a sequence of blocks (typically each block is 64KB in size, but this is configurable). A block index (stored at the end of the SSTable) is used to locate blocks; the index is loaded into memory when the SSTable is opened. A lookup can be performed with a single disk seek: we first find the appropriate block by performing a binary search in the in-memory index, and then reading the appropriate block from disk. Optionally, an SSTable can be completely mapped into memory, which allows us to perform lookups and scans without touching disk.

    3.2 ж•°жҚ®еҶҷе…ҘиҝҮзЁӢ¶

    еҶҷ MemTable ж—¶пјҢеҶҷе…Ҙж–°зҡ„ж•°жҚ®пјҢдёҺдҝ®ж”№зҺ°жңүж•°жҚ®зҡ„йғЁеҲҶеӯ—ж®өд»ҘеҸҠдҝ®ж”№зҺ°жңүж•°жҚ®зҡ„жүҖжңүеӯ—ж®өпјҢеҶҷе…Ҙж“ҚдҪңиҝҮзЁӢеҮ д№ҺжҳҜдёҖж ·зҡ„пјҢйғҪжҳҜжҠҠдј е…Ҙзҡ„ж•°жҚ®пјҲеҗҲе№¶пјүеҶҷе…ҘеҲ° MemTable дёӯгҖӮеҲ йҷӨж•°жҚ®ж—¶пјҢеҲҷжҳҜеңЁ MemTable дёӯеҶҷе…ҘдёҖжқЎеҲ йҷӨж Үи®°гҖӮеҪ“ MemTable зҡ„еӨ§е°ҸиҫҫеҲ°и®ҫе®ҡзҡ„еӨ§е°ҸпјҲе…ёеһӢеҖјжҳҜ 64KB пјүж—¶пјҢ LSM Tree дјҡжҠҠеҪ“еүҚ MemTable еҶ»з»“дёәдёҖдёӘдёҚеҸҜдҝ®ж”№зҡ„ Immutable MemTable пјҢз„¶еҗҺеҲӣе»әдёҖдёӘж–°зҡ„ MemTable дҫӣж–°зҡ„ж•°жҚ®еҶҷе…ҘгҖӮеҗҢж—¶пјҢ LSM Tree дёҖиҲ¬дјҡжңүдёҖдәӣдёҺеҶҷе…ҘзәҝзЁӢпјҲжҲ–иҝӣзЁӢпјүзӣёзӢ¬з«Ӣзҡ„иғҢжҷҜзәҝзЁӢпјҲжҲ–иҝӣзЁӢпјүиҙҹиҙЈе°Ҷ Immutable MemTable flush еҲ°зЈҒзӣҳдёӯпјҢе°Ҷж•°жҚ®жҢҒд№…еҢ–гҖӮе·Із»Ҹ flush еҲ°зЈҒзӣҳдёӯзҡ„ Immutable MemTable еҜ№еә”зҡ„ WAL е°ұеҸҜд»Ҙд»ҺзЈҒзӣҳдёӯеҲ йҷӨдәҶгҖӮиҖҢеҶ…еӯҳдёӯ Immutable MemTable ж•°йҮҸзҡ„еӨҡе°‘еӨ„еҶідәҺ Immutable MemTable flush зҡ„йҖҹеәҰдёҺ Immutable MemTable з”ҹжҲҗзҡ„йҖҹеәҰпјҲж•°жҚ®еҶҷе…ҘйҖҹеәҰпјүзҡ„е·®еҖј

    д»Һ LSM Tree ж•°жҚ®еҶҷе…ҘиҝҮзЁӢеҸҜзҹҘпјҢ LSM Tree ж•°жҚ®еҶҷе…Ҙзҡ„ж“ҚдҪңйқһеёёз®ҖеҚ•пјҢиҝҮзЁӢд№ҹйқһеёёе°‘пјҢеҸӘиҰҒеҶҷ WAL е’ҢеҶ…еӯҳдёӯзҡ„ MemTable еҚіеҸҜгҖӮиҖҢеҶҷ WAL жҳҜд»ҘеңЁж–Үд»¶жң«е°ҫиҝҪеҠ и®°еҪ•ж–№ејҸзҡ„йЎәеәҸеҶҷпјҢж— йңҖж“ҚдҪңд»»дҪ•ж•°жҚ®з»“жһ„пјҢеҶҷе…ҘйҖҹеәҰйқһеёёеҝ«гҖӮеҶҷ MemTable иҷҪз„¶д№ҹйңҖиҰҒиҝӣиЎҢи·іиЎЁзҡ„жҸ’е…Ҙе’ҢжҺ’еәҸж“ҚдҪңпјҢдҪҶжҳҜ MemTable жҳҜдёҖдёӘеҶ…еӯҳж•°жҚ®з»“жһ„пјҢеҗҢж—¶ MemTable зҡ„еӨ§е°ҸжҺ§еҲ¶еңЁдёҖдёӘйқһеёёйқһеёёе°Ҹзҡ„и§„жЁЎпјҲжҜ”еҰӮ 64KB пјүпјҢжүҖд»ҘеҶҷ MemTable д№ҹжҳҜдёҖдёӘйқһеёёйқһеёёеҝ«зҡ„иҝҮзЁӢгҖӮ

    еҗҢж—¶пјҢжҲ‘д»¬иҝҳеҸҜд»ҘзңӢеҲ°пјҢж•°жҚ®зҡ„еҶҷе…ҘйҖҹеәҰдёҺж•°жҚ®еә“дёӯж•°жҚ®жҖ»йҮҸзҡ„еӨҡе°‘жІЎжңүе…ізі»гҖӮиҖҢдё”пјҢдёҚз®ЎжҳҜеҶҷе…Ҙж–°зҡ„ж•°жҚ®пјҢиҝҳжҳҜе…ЁйҮҸдҝ®ж”№жҲ–йғЁеҲҶдҝ®ж”№зҺ°жңүж•°жҚ®пјҢжҲ–жҳҜеҲ йҷӨзҺ°жңүж•°жҚ®пјҢ LSM Tree зҡ„еҶҷе…ҘйҖҹеәҰд№ҹеҮ д№ҺжІЎеӨҡеӨ§е·®еҲ«гҖӮд№ҹе°ұжҳҜиҜҙпјҢ LSM Tree зҡ„еҶҷе…ҘйҖҹеәҰжҳҜзЁіе®ҡзҡ„пјҢи·ҹж•°жҚ®и§„жЁЎе’Ңж•°жҚ®жӣҙж–°зұ»еһӢйғҪжІЎжңүе…ізі»гҖӮ

    3.3 ж•°жҚ®жҹҘжүҫиҝҮзЁӢж №жҚ®¶

    LSM Tree зҡ„еҶҷе…Ҙзү№зӮ№жҲ‘д»¬зҹҘйҒ“пјҢеҰӮжһңдёҖйЎ№ж•°жҚ®жӣҙж–°еӨҡж¬ЎпјҢиҝҷйЎ№ж•°жҚ®еҸҜиғҪдјҡеӯҳеӮЁеңЁеӨҡдёӘдёҚеҗҢзҡ„ SSTable дёӯпјҢз”ҡиҮідёҖйЎ№ж•°жҚ®зҡ„дёҚеҗҢйғЁеҲҶзҡ„жңҖж–°ж•°жҚ®еҶ…е®№еӯҳеӮЁеңЁдёҚеҗҢзҡ„ SSTable дёӯпјҲж•°жҚ®йғЁеҲҶжӣҙж–°зҡ„еңәжҷҜпјүгҖӮ LSM Tree жҠҠиҝҷз§ҚзҺ°иұЎеҸ«еҒҡз©әй—ҙж”ҫеӨ§пјҲ space amplification пјүпјҢеӣ дёәдёҖйЎ№ж•°жҚ®еңЁзЈҒзӣҳдёӯеӯҳеӮЁдәҶеӨҡд»ҪеүҜжң¬пјҢиҖҢиҖҒзҡ„еүҜжң¬жҳҜе·Із»ҸиҝҮж—¶дәҶзҡ„пјҢдёҚйңҖиҰҒзҡ„пјҢж•°жҚ®е®һйҷ…еҚ з”Ёзҡ„еӯҳеӮЁз©әй—ҙжҜ”жңүж•Ҳж•°жҚ®йңҖиҰҒзҡ„еӨ§гҖӮ

    з©әй—ҙж”ҫеӨ§иҝҷз§ҚзҺ°иұЎеҜјиҮҙ LSM Tree зҡ„жҹҘжүҫиҝҮзЁӢжҳҜиҝҷж ·зҡ„пјҡжҢүж–°еҲ°иҖҒзҡ„йЎәеәҸжҹҘжүҫ SSTable пјҢзӣҙеҲ°еңЁжҹҗдёӘпјҲжҲ–жҹҗдәӣдёӘпјү SSTable дёӯжҹҘжүҫеҲ°дәҶжүҖйңҖзҡ„ж•°жҚ®пјҢжҲ–иҖ…жңҖиҖҒзҡ„ SSTable жҹҘжүҫе®Ңд№ҹжІЎжңүжүҫеҲ°йңҖиҰҒзҡ„ж•°жҚ®гҖӮе…·дҪ“жҹҘжүҫйЎәеәҸдёәпјҡе…ҲеңЁеҶ…еӯҳ MemTable дёӯжҹҘжүҫпјҢз„¶еҗҺеңЁеҶ…еӯҳдёӯзҡ„ Immutable MemTable дёӯжҹҘжүҫпјҢз„¶еҗҺеңЁ level 0 SSTable дёӯжҹҘжүҫпјҢжңҖеҗҺеңЁ level N SSTable дёӯжҹҘжүҫгҖӮ

    жҹҘжүҫжҹҗдёӘе…·дҪ“зҡ„ SSTable ж—¶пјҢдёҖиҲ¬е…ҲжҠҠ SSTable зҡ„е…ғж•°жҚ® block иҜ»еҲ°еҶ…еӯҳдёӯпјҢж №жҚ® BloomFilter еҸҜд»Ҙеҝ«йҖҹзЎ®е®ҡж•°жҚ®еңЁеҪ“еүҚ SSTable дёӯжҳҜеҗҰеӯҳеңЁпјҢеҰӮжһңеӯҳеңЁпјҢеҲҷйҮҮз”ЁдәҢеҲҶжі•зЎ®е®ҡж•°жҚ®еңЁе“ӘдёӘж•°жҚ® block пјҢз„¶еҗҺе°Ҷзӣёеә”ж•°жҚ® block иҜ»еҲ°еҶ…еӯҳдёӯиҝӣиЎҢзІҫзЎ®жҹҘжүҫгҖӮ

    д»Һ LSM Tree ж•°жҚ®жҹҘжүҫиҝҮзЁӢжҲ‘д»¬еҸҜд»ҘзңӢеҲ°пјҢдёәдәҶжҹҘжүҫеҲ°зӣ®ж Үж•°жҚ®пјҢжҲ‘д»¬йңҖиҰҒиҜ»еҸ–е№¶жҹҘжүҫдёҚеҢ…еҗ«зӣ®ж Үж•°жҚ®зҡ„ SSTable пјҢеҰӮжһңзӣ®ж Үж•°жҚ®еңЁжңҖеә•еұӮ level N зҡ„ SSTable дёӯпјҢжҲ‘д»¬йңҖиҰҒиҜ»еҸ–е’ҢжҹҘжүҫжүҖжңүзҡ„ SSTable пјҒ LSM Tree жҠҠиҝҷз§ҚиҜ»еҸ–е’ҢжҹҘжүҫдәҶж— е…і SSTable зҡ„зҺ°иұЎеҸ«еҒҡиҜ»ж”ҫеӨ§пјҲ read amplification пјүгҖӮ

    иҜ»ж”ҫеӨ§зҺ°иұЎдёҘйҮҚеҪұе“ҚдәҶ LSM Tree ж•°жҚ®жҹҘжүҫжҖ§иғҪпјҢз”ҡиҮіжҳҜзҒҫйҡҫжҖ§зҡ„пјҲж•°жҚ®еҺӢж №дёҚеӯҳеңЁжҲ–еңЁжңҖиҖҒзҡ„ SSTable дёӯпјүпјҢи®әж–ҮгҖҠBigTableгҖӢжҸҗеҲ°дәҶеҮ з§ҚжҸҗеҚҮж•°жҚ®жҹҘжүҫжҖ§иғҪзҡ„ж–№жі•пјҡ

    1. еҺӢзј©пјҲ Compression пјү

      д»Ӣз»Қ SSTable ж—¶е·Із»ҸжҸҗеҲ°дәҶпјҢеҜ№ж•°жҚ® block иҝӣиЎҢеҺӢзј©пјҢйҖҡиҝҮеўһеҠ еҚ з”Ё CPU еҺӢзј©е’Ңи§ЈеҺӢзј©иө„жәҗжқҘйҷҚдҪҺж•°жҚ® block зЈҒзӣҳз©әй—ҙеҚ з”Ёе’ҢиҜ»еҶҷж—¶й—ҙгҖӮ

    2. BloomFilter

      жҸҸиҝ°жҹҘжүҫиҝҮзЁӢж—¶д№ҹе·Із»ҸжҸҗеҲ°дәҶпјҢ BloomFilter еҸҜд»Ҙеҝ«йҖҹзЎ®е®ҡж•°жҚ®дёҚеңЁ SSTable дёӯпјҢиҖҢдёҚз”ЁиҜ»еҸ–ж•°жҚ® block еҶ…е®№

    3. зј“еӯҳпјҲ Cache пјү

      еӣ дёә SSTable жҳҜдёҚеҸҜеҸҳзҡ„пјҢйқһеёёйҖӮеҗҲзј“еӯҳеҲ°еҶ…еӯҳдёӯпјҢиҝҷж ·зғӯзӮ№ж•°жҚ®дёҚз”Ёи®ҝй—®зЈҒзӣҳ

    4. SSTable еҗҲе№¶пјҲ Compaction пјү

      е°ҶеӨҡдёӘ SSTable еҗҲе№¶дёәдёҖдёӘ SSTable пјҢеҲ йҷӨж—§ж•°жҚ®жҲ–зү©зҗҶеҲ йҷӨе·Із»Ҹиў«еҲ йҷӨзҡ„ж•°жҚ®пјҢйҷҚдҪҺз©әй—ҙж”ҫеӨ§пјӣеҗҢж—¶еҮҸе°‘ SSTable ж•°йҮҸпјҢйҷҚдҪҺиҜ»ж”ҫеӨ§

    е…¶е®һиҝҷдәӣдјҳеҢ–жҺӘж–ҪпјҢйҷӨдәҶ SSTable еҗҲе№¶жҳҜ LSM Tree зӢ¬жңүзҡ„пјҢеүҚйқўдёүжқЎйғҪжҳҜж•°жҚ®еә“йҖҡз”Ёзҡ„жҺӘж–ҪпјҢз”ҡиҮі SSTable еҗҲе№¶д№ҹдёҚжҳҜ LSM Tree зӢ¬жңүзҡ„пјҢе®ғдёҺжӣҙж—©еҮәзҺ°зҡ„ lucene зҡ„ж®өеҗҲе№¶пјҲ Segment Merge пјүзҡ„еҺҹзҗҶе’Ңзӣ®ж Үе…¶е®һжҳҜжңүзӣёдјјзҡ„ең°ж–№зҡ„пјҲеҪ“з„¶д»–д»¬зҡ„еҶҷе…Ҙе’ҢжҹҘжүҫиҝҮзЁӢиҝҳжҳҜжңүжң¬иҙЁеҢәеҲ«зҡ„пјүгҖӮдёӢйқўиҜҰз»Ҷд»Ӣз»ҚдёӢ SSTable еҗҲе№¶

    4 SSTableеҗҲе№¶¶

    д»ҺдёҠиҝ°иҜ»еҶҷиҝҮзЁӢжҸҸиҝ°еҸҜд»ҘзңӢеҲ°пјҢ LSM Tree иҷҪз„¶ж•°жҚ®еҶҷе…ҘйҖҹеәҰйқһеёёеҝ«пјҢдҪҶжҳҜеӯҳеңЁз©әй—ҙж”ҫеӨ§е’ҢиҜ»ж”ҫеӨ§зҡ„зҺ°иұЎпјҢиҝҷдәӣзҺ°иұЎеҰӮжһңдёҚиҝӣиЎҢжҠ‘еҲ¶пјҢеҸҜиғҪеҜјиҮҙиҜ»жҖ§иғҪзҡ„жһҒз«ҜжҒ¶еҢ–е’Ңз©әй—ҙеҚ з”ЁиҝҮдәҺиҶЁиғҖпјҢжңҖз»ҲеҜјиҮҙ LSM Tree еңЁе®һйҷ…з”ҹдә§зҺҜеўғдёӯдёҚеҸҜз”ЁгҖӮ SSTable еҗҲе№¶е°ұжҳҜз”ЁжқҘзј“и§Јиҝҷз§ҚзҺ°иұЎзҡ„гҖӮ LSM Tree ж”ҜжҢҒе°ҶеӨҡдёӘ SSTable еҗҲе№¶дёәдёҖдёӘж–°зҡ„ SSTable пјҢеҗҲе№¶иҝҮзЁӢдёӯпјҢдјҡеҲ йҷӨж—§зҡ„йҮҚеҸ ж•°жҚ®пјҢе№¶зңҹжӯЈзү©зҗҶеҲ йҷӨиў«еҲ йҷӨзҡ„ж•°жҚ®пјҢд№ҹеҮҸе°‘дәҶ SSTable зҡ„ж•°йҮҸпјҢиҝҷж ·ж¶ҲйҷӨдәҶз©әй—ҙж”ҫеӨ§пјҢеҗҢж—¶д№ҹжҸҗеҚҮдәҶж•°жҚ®жҹҘжүҫзҡ„жҖ§иғҪгҖӮдҪҶжҳҜпјҢеҗҲе№¶йңҖиҰҒе°ҶеҗҲе№¶ж¶үеҸҠзҡ„ SSTable иҜ»е…ҘеҶ…еӯҳпјҢе№¶жҠҠеҗҲе№¶еҗҺдә§з”ҹзҡ„ж–°зҡ„ SSTable еҶҷе…ҘзЈҒзӣҳпјҢдјҡеўһеҠ зЈҒзӣҳ IO е’Ң CPU зҡ„ж¶ҲиҖ—пјҢиҝҷз§ҚеҶҷе…ҘзЈҒзӣҳзҡ„ж•°жҚ®йҮҸеӨ§дәҺе®һйҷ…ж•°жҚ®йҮҸзҺ°иұЎжҲҗдёәеҶҷж”ҫеӨ§пјҲ write amplification пјүпјҢеҪ“з„¶еҗҲе№¶д№ҹж„Ҹе‘ізқҖиҜ»ж”ҫеӨ§гҖӮз”ұжӯӨеҸҜи§ҒпјҢ SSTable еҗҲе№¶жҳҜдёҖжҠҠеҸҢеҲғеү‘пјҢжңүеҲ©д№ҹжңүејҠпјҢйңҖиҰҒеҗҲзҗҶеҲ©з”ЁгҖӮ

    SSTable еҗҲе№¶еҲҶдёә minor compaction е’Ң major compaction пјҢ minor compaction жҳҜжҢҮе°ҶеҶ…еӯҳдёӯзҡ„ Immutable MemTable еҶ…е®№ flush еҲ°зЈҒзӣҳдёӯеҪўжҲҗ SSTable ж—¶иҝӣиЎҢзҡ„ж•°жҚ®еӨ„зҗҶиҝҮзЁӢгҖӮ major compaction еҲҷжҳҜжҢҮзӣёйӮ»зҡ„дёӨзә§ level е°Ҷж•°еӯ—е°Ҹ level пјҲеҰӮ level 0 пјүзҡ„ SSTable еҗҲе№¶е…Ҙж•°еӯ—еӨ§ level пјҲеҰӮ level 1 пјүзҡ„ SSTable дёӯзҡ„иҝҮзЁӢгҖӮ

    SSTable еҗҲе№¶е…¶е®һе°ұжҳҜеңЁз©әй—ҙж”ҫеӨ§гҖҒеҶҷж”ҫеӨ§гҖҒиҜ»ж”ҫеӨ§еҮ дёӘзӣёдә’еҲ¶зәҰзҡ„еӣ зҙ й—ҙеҜ»жұӮе№іиЎЎпјҢдёҚеҗҢзҡ„еә”з”ЁеңәжҷҜйңҖиҰҒйҮҚзӮ№дјҳеҢ–и§ЈеҶіжҹҗдёӘй—®йўҳпјҢд»ҺиҖҢеҪўжҲҗдәҶеҮ з§Қе…ёеһӢзҡ„ SSTable еҗҲе№¶зӯ–з•Ҙпјҡ

    4.1 leveled compaction¶

    leveled compaction дёәжҜҸеұӮ level зҡ„ SSTable ж•°жҚ®жҖ»еӨ§е°Ҹи®ҫзҪ®дёҖдёӘйҳҲеҖјпјҢ level ж•°и¶ҠеӨ§пјҢйҳҲеҖји®ҫзҪ®зҡ„д№ҹи¶ҠеӨ§пјҢжҜ”еҰӮ level0 йҳҲеҖјдёә 10MB гҖҒ level1 йҳҲеҖји®ҫзҪ®дёә 100MB гҖҒ level2 йҳҲеҖји®ҫзҪ®дёә 1000MB гҖӮеҪ“жҹҗ level еұӮзҡ„ж•°жҚ®жҖ»йҮҸеӨ§е°Ҹи¶…иҝҮи®ҫзҪ®зҡ„йҳҲеҖјж—¶пјҢеҲҷйҖүеҸ–дёҖдёӘ SSTable еҗҲе№¶е…Ҙй«ҳдёҖзә§ level еұӮзҡ„дёҖдёӘжҲ–еӨҡдёӘ SSTable дёӯгҖӮй«ҳ level еұӮж¶үеҸҠзҡ„ SSTable зҡ„йҖүжӢ©еӨ„еҶідәҺж•°жҚ®зҡ„еҲҶеёғпјҢд»ҘеҗҲе№¶еҗҺй«ҳ level еұӮдёӯзҡ„жүҖжңү SSTable ж•°жҚ®жҳҜж•ҙдҪ“жңүеәҸзҡ„дёәеҮҶпјҲ one sorted run пјүпјҢд№ҹе°ұжҳҜиҜҙж•°жҚ®еңЁеҗҢдёҖеұӮдёӯдёҚеӯҳеңЁйҮҚеҸ зҡ„зҺ°иұЎгҖӮ

    leveled compaction еӣ дёәдјҡе°Ҷ SSTable еҗҲе№¶е…ҘеӨҡдёӘ SSTable дёӯпјҢиғҪдҝқиҜҒеҗҢдёҖеұӮзҡ„ SSTable дёӯж•°жҚ®дёҚдјҡйҮҚеҸ пјҢжүҖд»ҘжҳҜдёҖз§ҚжңҖе°ҸеҢ–з©әй—ҙж”ҫеӨ§зҡ„зӯ–з•ҘгҖӮдҪҶжҳҜпјҢеҗҲе№¶ж—¶йңҖиҰҒиҜ»еҸ–еӨҡдёӘ SSTable пјҢз„¶еҗҺеҶҷе…ҘеӨҡдёӘжӣҙж–°еҗҺзҡ„ SSTable пјҢеҜјиҮҙжӣҙеӨ§зҡ„еҶҷж”ҫеӨ§е’ҢиҜ»ж”ҫеӨ§гҖӮдҪҶжҳҜеңЁж•°жҚ®жҹҘжүҫиҝҮзЁӢдёӯпјҢз”ұдәҺеҮҸе°‘дәҶйңҖиҰҒжҹҘжүҫ SSTable зҡ„ж•°йҮҸпјҢйҷҚдҪҺдәҶж•°жҚ®жҹҘжүҫж—¶зҡ„иҜ»ж”ҫеӨ§пјҢжҸҗеҚҮдәҶж•°жҚ®жҹҘжүҫзҡ„жҖ§иғҪгҖӮжүҖд»ҘпјҢ leveled compaction йҖӮеҗҲжҜ”иҫғе…іжіЁж•°жҚ®жҹҘиҜўйҖҹеәҰе’ҢжҺ§еҲ¶зЈҒзӣҳз©әй—ҙеҚ з”Ёзҡ„еңәжҷҜгҖӮжҜ”еҰӮпјҢж•°жҚ®еҶҷе…ҘдёҖж¬ЎпјҢдҪҶжҳҜдјҡиў«йў‘з№ҒеҸҚеӨҚжҹҘиҜўпјӣж•°жҚ®з»Ҹеёёиў«дҝ®ж”№пјҢдҪҶжҳҜйңҖиҰҒжҺ§еҲ¶зЈҒзӣҳз©әй—ҙеӨ§е°Ҹе’ҢдҝқиҜҒж•°жҚ®жҹҘиҜўйҖҹеәҰгҖӮ

    Cassandra зҡ„ LCS пјҲ Leveled Compaction Strategy пјүе’Ң RocksDB зҡ„ Classic Leveled Compaction йҮҮз”Ёзҡ„е°ұжҳҜ leveled compaction зӯ–з•ҘгҖӮ

    4.2 tiered compaction¶

    tiered compaction еҪ“жҹҗ level еұӮзҡ„ SSTable ж•°йҮҸиҫҫеҲ°и®ҫе®ҡзҡ„йҳҲеҖјж—¶пјҢеҲҷе°ҶиҜҘеұӮзҡ„еӨҡдёӘ SSTable еҗҲе№¶дёәдёҖдёӘж–°зҡ„ SSTable пјҢе№¶ж”ҫе…Ҙй«ҳдёҖеұӮ level дёӯгҖӮ

    tiered compaction жҜ”иҫғеҒҸйҮҚдәҺжҺ§еҲ¶еҶҷж”ҫеӨ§еңЁдёҖе®ҡзЁӢеәҰзҡ„еүҚжҸҗдёӢйҷҚдҪҺз©әй—ҙж”ҫеӨ§е’ҢжҸҗеҚҮж•°жҚ®жҹҘиҜўжҖ§иғҪгҖӮдҪҶжҳҜеңЁеҗҢдёҖеұӮзҡ„ SSTable дёӯиҝҳжҳҜеӯҳеңЁж•°жҚ®йҮҚеҸ пјҢж•°жҚ®жҹҘиҜўжҖ§иғҪдёҚеҰӮ leveled compaction гҖӮе°Өе…¶жҳҜйҡҸзқҖ level ж•°и¶ҠжқҘи¶ҠеӨ§пјҢеҚ•дёӘ SSTable ж•°жҚ®йҮҸд№ҹи¶ҠжқҘи¶ҠеӨ§пјҢеҗҲе№¶и§ҰеҸ‘жқЎд»¶д№ҹи¶ҠжқҘи¶ҠйҡҫпјҢиҝҷдәӣе·ЁеһӢ SSTable дёӯзҡ„йҮҚеҸ ж•°жҚ®е’Ңиў«еҲ йҷӨзҡ„ж•°жҚ®еҚ з”Ёзҡ„з©әй—ҙд№ҹе°ұи¶ҠжқҘи¶Ҡйҡҫиў«йҮҠж”ҫжҺүгҖӮ tiered compaction жҜ”иҫғйҖӮеҗҲж•°жҚ®дҝ®ж”№йў‘зҺҮдёҚй«ҳпјҢдё”жңҖиҝ‘еҶҷе…Ҙзҡ„ж•°жҚ®жҹҘиҜўйў‘зҺҮжҜ”иҫғй«ҳзҡ„еңәжҷҜгҖӮ

    Cassandra зҡ„ STCS пјҲ Size Tiered Compaction Strategy пјүе’Ң RocksDB зҡ„ universal Compaction дҪҝз”Ёзҡ„жҳҜ tiered compaction зӯ–з•ҘгҖӮ

    4.3 leveled-tiered mixed compaction¶

    з»“еҗҲ leveled compaction е’Ң tiered compaction зҡ„дјҳеҠҝпјҢеңЁйғЁеҲҶ level д№Ӣй—ҙйҮҮз”Ё tiered compaction пјҢеңЁеҸҰдёҖйғЁеҲҶ level д№Ӣй—ҙйҮҮз”Ё leveled compaction гҖӮ

    RocksDB зҡ„ leveled Compaction йҮҮз”Ёзҡ„жҳҜ leveled-tiered mixed compaction зӯ–з•ҘгҖӮж•°жҚ®д»Һ Immutable MemTable flush еҲ° level 0 ж—¶йҮҮз”Ёзҡ„жҳҜ tiered compaction пјҢд№ҹе°ұжҳҜиҜҙ level0 дёӯжҳҜеӯҳеңЁйҮҚеҸ ж•°жҚ®зҡ„гҖӮ level0 еҲ° levelN д№Ӣй—ҙйҮҮз”Ё leveld compaction зӯ–з•ҘгҖӮ

    4.4 FIFO compaction¶

    FIFO compaction еңЁзЈҒзӣҳдёӯеҸӘжңүдёҖдёӘ level еұӮзә§пјҢ SSTable жҢүз”ҹжҲҗзҡ„ж—¶й—ҙйЎәеәҸжҺ’еҲ—пјҢеҲ йҷӨиҝҮж—©з”ҹжҲҗзҡ„ SSTable гҖӮ

    FIFO compaction йҖӮеҗҲж—¶й—ҙеәҸеҲ—ж•°жҚ®пјҢдёҖж—Ұз”ҹжҲҗж•°жҚ®е°ұдёҚдјҡеҶҚдҝ®ж”№гҖӮ

    Cassandra зҡ„ TWCS пјҲ Time Window Compaction Strategy пјүгҖҒ DTCS пјҲ Date Tiered Compaction Strategy пјүе’Ң RocksDB зҡ„ FIFO compaction дҪҝз”Ёзҡ„жҳҜ FIFO зұ»еһӢзҡ„еҗҲе№¶зӯ–з•ҘгҖӮ

    5 LSM TreeеҺҹзҗҶеҜ№зі»з»ҹиҜ»еҶҷжҖ§иғҪжҸҗеҚҮзҡ„еҮ зӮ№еҗҜзӨә¶

    1. еҶҷж•°жҚ®ж—¶е°ҪйҮҸжү№йҮҸж“ҚдҪңгҖӮ LSM Tree ж•°жҚ®еҶҷе…ҘжҖ§иғҪе·Із»ҸеҫҲй«ҳдәҶпјҢдҪҶжҳҜжү№йҮҸж“ҚдҪңж—¶еҸҜд»ҘиҠӮзңҒзҪ‘з»ңдј иҫ“ RTT ж—¶й—ҙгҖӮ

    2. е°Ҷж•°жҚ®иҝӣиЎҢеҲҶзүҮгҖӮиҝҷж ·еӨҡдёӘеҲҶзүҮеҸҜд»Ҙе№¶иЎҢеҶҷпјҢеҰӮжһңж•°жҚ®и·Ҝз”ұеӨ„зҗҶеҫ—еҪ“пјҢд№ҹеҸҜд»ҘжҸҗеҚҮж•°жҚ®жҹҘиҜўйҖҹеәҰгҖӮдҪҶжҳҜеўһеҠ дәҶз»ҙжҠӨеӨҡдёӘеҲҶзүҮж•°жҚ®иҜ»еҶҷзҡ„еӨҚжқӮеәҰгҖӮ

    3. йҖүжӢ©еҗҲйҖӮзҡ„дё»й”®гҖӮдёӨдёӘжҜ”иҫғжөҒиЎҢзҡ„йҖүжӢ©жҳҜпјҡ

      1. йҮҮз”ЁйҖ’еўһзҡ„ж•°еӯ—дҪңдёәдё»й”®пјӣ

      2. йҮҮз”ЁдёҡеҠЎжң¬иә«ж ҮиҜҶз¬ҰдҪңдёәдё»й”®гҖӮж•°еӯ—дҪңдёәдё»й”®еҸҜд»ҘеҮҸе°‘еҶҷе…Ҙе’ҢжҹҘиҜўж—¶иҝӣиЎҢжҜ”иҫғгҖҒжҺ’еәҸзӯүж“ҚдҪңзҡ„ж—¶й—ҙпјҢиҝҳиғҪжҸҗеҚҮзҙўеј•зј“еӯҳзҡ„ж•ҲзҺҮпјӣйҖ’еўһзҡ„ж•°еӯ—еҫҖеҫҖдҝқиҜҒжҳҜйЎәеәҸеҶҷе…Ҙзҡ„пјҢеҸҜд»ҘеҮҸе°‘жҺ’еәҸж—¶й—ҙгҖӮдҪҶжҳҜйҖ’еўһзҡ„ж•°еӯ—еҫҖеҫҖдёҚе…·еӨҮдёҡеҠЎиҜӯд№үпјҢдёҡеҠЎе®һйҷ…жҹҘиҜўж—¶йңҖиҰҒе…ҲжҹҘдәҢзә§зҙўеј•пјҢз„¶еҗҺиҝӣиЎҢдё»й”®жҹҘжүҫгҖӮдёҡеҠЎж ҮиҜҶз¬ҰеҫҖеҫҖжҳҜдёҡеҠЎе®һйҷ…зҡ„иә«д»ҪеҢәеҲҶз¬ҰеҸ·пјҢдёҡеҠЎд№ҹеҫҖеҫҖйҖҡиҝҮдёҡеҠЎж ҮиҜҶз¬ҰиҝӣиЎҢж•°жҚ®жҹҘиҜўгҖӮдҪҶжҳҜдёҡеҠЎж ҮиҜҶз¬ҰеҫҖеҫҖжҳҜдёҖдёӘеӯ—з¬ҰдёІпјҢеҸҜиғҪдјҡжҜ”иҫғй•ҝпјҢиҝҷж ·жҜ”иҫғгҖҒжҺ’еәҸгҖҒзј“еӯҳж•ҲзҺҮж–№йқўдёҚеҰӮж•°еӯ—гҖӮдёҖиҲ¬жғ…еҶөдёӢпјҢжң¬дәәе»әи®® LSM Tree ж•°жҚ®еә“йҮҮз”ЁдёҡеҠЎж ҮиҜҶз¬ҰдҪңдёәдё»й”®гҖӮеӣ дёәдёәдёҡеҠЎж ҮиҜҶз¬Ұе»әз«Ӣзҙўеј•д»ҘеҸҠз»ҙжҠӨзҙўеј•зҡ„жҲҗжң¬жҳҜе…ҚдёҚдәҶзҡ„пјҢдёҺе…¶е»әз«ӢдәҢзә§зҙўеј•пјҢдёҚеҰӮзӣҙжҺҘе»әз«Ӣдё»й”®зҙўеј•гҖӮ

    4. и®ҫи®ЎеҗҲзҗҶзҡ„дәҢзә§зҙўеј•пјҢдёҚе»әз«ӢдёҚйңҖиҰҒзҡ„дәҢзә§зҙўеј•гҖӮдәҢзә§зҙўеј•еҸҜд»ҘжҸҗеҚҮзӣёеә”ж•°жҚ®зҡ„жҹҘиҜўйҖҹеәҰпјҢжҜҸеўһеҠ дёҖдёӘдәҢзә§зҙўеј•пјҢе°ұйңҖиҰҒйўқеӨ–з»ҙжҠӨзӣёеә”зҡ„дәҢзә§зҙўеј•ж–Үд»¶пјҢдёҘйҮҚеҪұе“ҚеҶҷе…Ҙж•°жҚ®жҖ§иғҪгҖӮ

    5. ж №жҚ®е…·дҪ“еңәжҷҜдҪҝз”ЁеҗҲйҖӮзҡ„ SSTable еҗҲе№¶зӯ–з•ҘгҖӮеҚ•ж¬ЎеҶҷпјҢйў‘з№ҒиҜ»еңәжҷҜйҖүжӢ© leveled compaction зӯ–з•ҘгҖӮ

    6. еңЁе…Ғи®ёжғ…еҶөдёӢе…ій—ӯиҮӘеҠЁ SSTable еҗҲе№¶пјҢеңЁдёҡеҠЎйҮҸдҪҺзҡ„ж—¶й—ҙж®өејәеҲ¶жү§иЎҢ SSTable еҗҲе№¶гҖӮ

    7. ж•°жҚ®жӣҙж–°ж—¶еҗҲзҗҶйҖүжӢ©е…ЁйҮҸжӣҙж–°пјҲиҰҶзӣ–еҶҷпјүж–№ејҸиҝҳжҳҜйғЁеҲҶжӣҙж–°пјҲеўһйҮҸеҶҷпјүж–№ејҸгҖӮе…ЁйҮҸжӣҙж–°ж–№ејҸеўһеҠ дәҶдј иҫ“е’ҢеҶҷе…Ҙзҡ„ж•°жҚ®йҮҸпјҢдҪҶжҳҜеҸҜд»ҘжҸҗеҚҮж•°жҚ®жҹҘиҜўйҖҹеәҰгҖӮйғЁеҲҶжӣҙж–°ж–№ејҸдјҡдҪҝж•°жҚ®еҲҶеёғеңЁеӨҡдёӘ SSTable дёӯпјҢйңҖиҰҒжҹҘиҜўе’ҢеҗҲе№¶еӨҡдёӘ SSTable дёӯзҡ„ж•°жҚ®жүҚиғҪеҫ—еҲ°е®Ңж•ҙзҡ„ж•°жҚ®пјҢдјҡйҷҚдҪҺж•°жҚ®жҹҘиҜўйҖҹеәҰгҖӮеҰӮжһңж•°жҚ®дҝ®ж”№жҜ”иҫғйў‘з№ҒпјҢдё”йңҖиҰҒиҫғй«ҳжҹҘиҜўйҖҹеәҰпјҢе»әи®®йҮҮз”Ёе…ЁйҮҸжӣҙж–°ж–№ејҸгҖӮ
